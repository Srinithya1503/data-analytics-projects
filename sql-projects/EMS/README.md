# SQL Capstone Project – Multi-Phase Database Systems

## Overview
This project is a three-phase SQL Server implementation designed for freshers
to understand real-world database concepts including data modeling, querying,
procedures, and access control.

---

## Phase 1: Employee Management System
**Concepts Covered**
- Primary & Foreign Keys
- Subqueries
- Aggregate functions
- User-defined functions

**Outcome**
- Retrieve employees with above-average salary
- Filter employees by department
- Generate formatted employee names

---

## Phase 2: Sales Analysis System
**Concepts Covered**
- Views for reporting
- Stored procedures
- Business rule enforcement

**Outcome**
- Centralized sales reporting view
- Safe product price updates
- Reusable analytics layer

---

## Phase 3: Access Management System
**Concepts Covered**
- Server roles vs Database roles
- Least privilege principle
- Enterprise security design

**Outcome**
- Admins manage servers without data access
- Developers modify schema only
- BI team reads analytical data
- Support team accesses customers safely

---

## Technologies Used
- SQL Server (T-SQL)
- AdventureWorks Sample Database
