# Data Assumptions & Identified Gaps

## Identified Data Gaps
- No refund data present in raw transactional tables
- No explicit customer-level identifiers for retention analysis
- Limited product catalog (4 SKUs)

---

## Mitigation Strategy
- Introduced synthetic refund records (5% rate)
- Clearly labeled all synthesized data
- Avoided overstating retention or lifetime value

---

## Analytical Risks Prevented
- Overestimated profitability
- Misleading CAC calculations
- False conclusions about operational efficiency
